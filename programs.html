About Programs

  • VisualBasic .NET software:
    • zip.exe:
      • The zip.exe program runs only if arguments are included:
      • Extract: zip.exe -extract [zip-file] [destination-folder]
      • Compress: zip.exe -compress [source-folder] [output-zip]
      • Help: zip.exe -help
      • [NOTE: Use ` instead of spaces in filenames.]
